================================================================================
  TROUBLESHOOTING
================================================================================

Animations not working?
  → Check 'use client' at top of component
  → Verify motion is imported from 'framer-motion'
  → Use motion.div not regular div

Styles not applying?
  → Check CSS in globals.css
  → Verify custom colors in :root
  → Use className not class

Page not updating?
  → Check useState is imported
  → Verify setCurrentStep is called
  → Look for console errors (F12)

Mobile looks weird?
  → Check responsive classes (sm:, md:)
  → Verify viewport meta tag
  → Test in Chrome DevTools mobile emulation
